# PHP LOGIN SYSTEM

## Sviluppo di un sistema di login in PHP

- versione procedurale
- versione object oriented
- versione con PDO

Dopo aver clonato il repository in locale, prima di eseguire il codice su un qualsiasi LAMP STACK assicurarsi
